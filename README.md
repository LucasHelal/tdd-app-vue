# tdd-app

## Configuração do projeto 


### Pré-Requisitos

- [Node](https://nodejs.org/en/download/)
```
curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
```
```
sudo apt-get install -y nodejs
```
- Yarn
  
```
npm install -g yarn
```

### Dependências

```
yarn
```


### Executar os testes
```
yarn test:unit


yarn test:unit:watch
```
